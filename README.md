RideMates - A Student-Centric Ride Sharing Platform
Developed to solve campus parking and reduce traffic congestion by enabling safe and affordable ridesharing among college students.
Overview
RideMates is a ride-sharing web app created exclusively for college students. By verifying users through their college IDs, RideMates ensures a safe, trusted community for ride sharing. It offers a cost-effective alternative to commercial ride-sharing services, promotes social connections, and helps alleviate campus parking challenges.

Features
Secure student registration with ID verification

Affordable rides compared to commercial options

Simple and intuitive UI for requesting and offering rides

Dashboard displaying current ride requests

User profile management and ride history

Integration with Maps API for route planning and fare estimation

Real-time communication via Twilio API notifications

Technologies Used
Backend: Node.js, Express.js

Frontend: HTML, CSS, Bootstrap, jQuery, EJS templates

Database: MongoDB

Authentication: Passport.js

External APIs: Maps API, Twilio API
